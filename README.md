# Wallpaper Changer

A simple wallpaper changer for windows.

# Features

1. Set predetermined wallpapers
2. Set custom wallpaper from path to image
3. Create wallpaper slideshow
more coming soon :)

# Installation

1. Clone this repo.
2. Install Python
3. If needed install ctypes, os, tkinter with pip.
4. Run main.py

# Usage

Well there probably isnt a reason for a normal person to use this but i wanted to make a simple wallpaper changer :)
